🎴 Vocab Flashcards - Help

Practice Mode

  • Click a direction button to choose how you want to practice:
    • Random: Random language pair each card
    • ES ↔ ZH: Spanish ↔ Chinese
    • ES ↔ EN: Spanish ↔ English
    • ES ↔ DE: Spanish ↔ German
    • ZH ↔ EN: Chinese ↔ English
    • ZH ↔ DE: Chinese ↔ German
    • EN ↔ DE: English ↔ German
    • Sequential: Click to flip cycles through all languages for the same word
  • Click the flashcard to flip and reveal the answer
  • Click ✓ Right if you knew it, ✗ Wrong if you didn't

Adding Words

  • Go to "Add Words" tab
  • Fill in Spanish, Chinese, Pinyin, English (German is optional)
  • Click "Add Word"

Importing

  • Go to "Import" tab
  • Paste CSV/TSV data directly, OR
  • Upload an Excel (.xlsx) file directly from the Import tab
  • Use command line: python import_vocab.py yourfile.xlsx
  • Supported formats: CSV (comma or semicolon), Excel (.xlsx)

Syncing with Colleagues

  • The vocabulary is stored in vocab-app/data/vocab.json
  • To share with colleagues:
    • git add vocab-app/data/vocab.json
    • git commit -m "Add new words"
    • git push
  • To get updates:
    • git pull

Progress Tracking

  • Each word has a proficiency level (0-5)
  • Clicking ✓ Right increases proficiency
  • Clicking ✗ Wrong decreases proficiency
  • Words with lower proficiency appear more often in random mode
